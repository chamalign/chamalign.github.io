# Preness Development

New development folder. Development proceeds with directory structure based on `architecture.md`.

## Directory Structure

```
preness-devs/
├── backend/              # Backend
│   ├── ai/              # 【Sasaki】GPT API Related
│   └── app/             # 【Vannes】API & DB
├── frontend/            # 【Vannes】Frontend
├── infrastructure/      # 【Horiba】Infrastructure & External Service Integration
├── docs/                # Documentation
├── scripts/            # Scripts
└── generated_exams/     # Generated exam storage (for development and testing)
```

## Development Process

1. Create branches for each phase
2. Create Pull Requests for review
3. Merge to main branch after approval

For details, refer to `docs/development/architecture.md` and `docs/development/development_roles.md`.
